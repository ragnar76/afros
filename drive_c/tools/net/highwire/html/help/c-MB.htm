Menu: File

Open...  ^O
Opens a file selector to load a local file. If Shift was pressed at the same time it will open a dialog box to enter a URL instead.
Info...  ^I
Shows an info box of the active frame of a frameset or of a page.
Quit     ^Q
Quits HighWire immediately.
 

Menu: View

Reload        ^R, F5
Reloads the current page or frameset.
Windows-1252   1
Reloads the current page or frameset with default character set ISO/IEC 8859-1, Latin-1, Windows codepage 1252. This is the default character set for any HTML browser.
ISO 8859-2     2
Reloads the current page or frameset with default character set ISO/IEC 8859-2, Latin-2.
ISO 8859-15    F
Reloads the current page or frameset with default character set ISO/IEC 8859-15, Latin-8.
Unicode UTF-8  U
Reloads the current page or frameset with default encoding Unicode 8-bit coded.
Unicode UTF-16
Reloads the current page or frameset with default encoding Unicode 16-bit coded (plain text only).
Unicode UTF-16LE
Reloads the current page or frameset with default encoding Unicode 16-bit little-endian coded (plain text only).
Mac Roman      M
Reloads the current page or frameset with default character set Apple Macintosh Roman.
Atari          A
Reloads the current page or frameset with default character set Atari ST system font.
Atari NVDI     N
(not available yet).
bigger         +
Increases the fontsize and reloads the current page or frameset.
smaller        -
Decreases the fontsize and reloads the current page or frameset.
 

Menu: History

A per-window list of previous visited pages, sorted top to down.
 

Menu: Options

Logging to Console   F7
Toggles debug logging to console.
Suppress Images      F8
Toggles displaying of image contents or their ALT textes.
Force Frame Controls
If selected, frames and framesets always get borders and are resizeable.
Cookies allowed
Indicates if cookies are sent and received.
