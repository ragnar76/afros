4.6 Freies Bild ändern

In einer Resource können neben Objektbäumen und freien Strings auch freie Images abgelegt werden. Dabei handelt es sich um einfarbige Bildchen, die z.B. für Mauszeiger o.ä. verwendet werden können. Man trifft sie allerdings eher selten an. Im Free-Image-Editor wird lediglich die Bildfarbe eingestellt. Mit "Ändern" kommt man in den Bild-Editor.

Im Programm kann das Bild wie folgt verwendet werden: 

   BITBLK **bitblk;
   rsrc_gaddr(16, ImageIndex, &bitblk);

Zu beachten ist hierbei, daß die Funktion rsrc_gaddr einen Zeiger auf einen Zeiger, der dann auf die Bildinformation zeigt, liefert.
