3.1.4 Sichern als

Die aktuelle Resourcedatei wird gesichert. Aktuell heisst in diesem Zusammenhang entweder die selektierten Dateien auf dem Desktop oder die Datei die zum obersten Fenster gehört. Im Gegensatz zum Menüpunkt 'Sichern' erscheint vorher ein Fileselector in dem ein neuer Name gewählt werden kann. Der neue Name wird anschliessend für diese Datei übernommen.
