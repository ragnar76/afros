2 Desktop

Auf dem Desktop liegen die Symbole für die geladenen Dateien, Bilder, Programme sowie für die Laufwerke und den Papierkorb. Am unteren Rand wird eine Funktionsleiste dargestellt, die über Einstellungen/Sonstiges jedoch auch abgestellt werden kann. Der Desktop wird normalerweise auf dem Hintergrund dargestellt, kann aber auch in ein Fenster gelegt oder ganz abgeschaltet werden.
