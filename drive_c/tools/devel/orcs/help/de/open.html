3.1.1 Öffnen

Dieser Menüpunkt kann verschiedene Aktionen auslösen.

  1. Sind auf dem Desktop ein oder mehrere Icons selektiert, werden sie geöffnet als ob man einen Doppelklick darauf gemacht hätte.
     

  2. Ist in dem obersten Fenster ein Objekt selektiert, wird es geöffnet als ob ein Doppelklick darauf gemacht worden wär.
     

  3. Trifft keine dieser Bedingungen zu, wird der Fileselector aufgerufen um eine Resource-Datei zu laden (Selectric, Boxkite, Freedom und alle anderen Fileselectoren die einen entsprechenden Cookie anlegen werden unterstützt).
