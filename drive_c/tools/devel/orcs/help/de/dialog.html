4.2 Dialogeditor

Im Dialog-Editor können Bäume vom Typ Dialog bearbeitet werden. Das aktuelle Objekt wird dabei jeweils durch einen kleinen schwarzen Kasten in der rechten unteren Ecke markiert, ausserdem werden einige Daten in der Infozeile des Fensters dargestellt (was dort dargestellt wird kann unter Einstellungen/Objektdaten festgelegt werden). Durch drücken und festhalten der linken Maustaste kann man Objekte verschieben oder vergrössern (indem man in eine der Ecken des Objekts klickt). Wird ein Objekt in ein anderes Fenster verschoben, kann man durch drücken einer Shifttaste festlegen ob es verschoben oder kopiert werden soll. Ein einfacher Klick lässt ein Popup-Menü erscheinen, ein Doppelklick führt zum Dialog zum bearbeiten des Objektes.

Im Dialog-Fenster sind ausserdem folgende Tasten belegt:

Esc Zeichnet das Fenster neu.
 
Ctrl-Cursor-Links Zeigt den vorherigen Baum im selben Fenster an.
 
Ctrl-Cursor-Rechts Zeigt den nächsten Baum im selben Fenster an.
