4 Resource Bäume

Wenn eine neue Datei geladen wird, wird zunächst einmal dieses Fenster angezeigt, in dem alle Bäume der Resource dargestellt werden. Wird ein Symbol für einen Dialog oder Menü angeklickt, erscheint ein Dialog in dem man Name und Typ ändern kann. Ein Doppelklick öffnet das Fenster zum Bearbeiten des entsprechenden Baums. Wenn die Shift-Taste beim selektieren festgehalten wird, kann man auch mehrere Bäume selektieren, die dann in andere Fenster verschoben oder kopiert werden können.
