4.2.1.3 String Objekt ändern

Neben den Standard-Objekt-Editor-Funktionen bietet der Button-Editor lediglich die Möglichkeit einen Text eingeben zu können. Es können maximal 255 Zeichen eingegeben werden. Wird das Zeilenende erreicht, wird gescrollt.
