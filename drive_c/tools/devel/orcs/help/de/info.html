3.1.7 Information

Dieser Menüpunkt kann (wie "Öffnen") verschiedene Aktionen auslösen.

  1. Sind auf dem Desktop ein oder mehrere Icons selektiert, wird die Information zu den entsprechenden Objekten angezeigt.
     

  2. Ist in dem obersten Fenster ein Objekt selektiert, wird die Information darüber angezeigt.
     

  3. Ist ein Fenster offen, wird die Information zum zugehörigen Inhalt angezeigt.
     

  4. Ansonsten wird die Programminformation angezeigt.
