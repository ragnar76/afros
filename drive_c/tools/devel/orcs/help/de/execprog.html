3.4.10 Programm starten

Dieser Menüpunkt lässt einen Fileselektor erscheinen, mit dem man ein Programm auswählen kann das gestartet werden soll. Häufig benutzte Programme können auch per Programm laden als Icon auf dem Desktop abgelegt und von dort gestartet werden.
