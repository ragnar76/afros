3.1.2 Neue Datei

Dieser Menüpunkt erzeugt (wie könnte es anders sein) eine neue leere Resourcedatei ohne Namen. Beim sichern einer solchen Datei wird dann gefragt wohin sie geschrieben werden soll. Eine neue Datei erhält die Eigenschaften die unter dem Menüpunkt Einstellungen eingestellt wurden.
