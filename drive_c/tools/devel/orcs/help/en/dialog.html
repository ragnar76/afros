4.2 Dialog-Editor

In the dialog editor trees of type dialog can be edited. The current object is marked by a small black box in the lower right corner in each case, also some data is shown in the info line of the window (what is shown there can be defined with Settings/Object data). By pressing and holding the left mouse button, you can move or enlarge objects (by clicking in one of the corners of the object). If an object is moved to another window, you can specify whether to move or copy it by pressing a shift key. A single click will let a pop-up menu appear, a double-click opens the dialog to edit the object.

In the dialog window also following keys are used:

Esc Redraws the window.
 
Ctrl-Cursor-Left Shows the previous tree in the same window.
 
Ctrl-Cursor-Rechts Shows the next tree in the same window.
