3.4.4 Default Output files

Use this dialog box to set the output files for newly created resource files (the settings for already loaded resources can be changed with File/Output files).

On the left are the output formats for various programming languages, on the right side the formats for various other RSC editors. RSD is the format used by Kuma NRSC, DEF is used by older Atari programs, DFN the format used by newer Atari programs (RCS2) and HRD is eg. used by WERCS and Interface. RSM is used by RsourceMaster. RSO is the format ORCS always uses and can not be disabled. All these formats can be both read and written.
