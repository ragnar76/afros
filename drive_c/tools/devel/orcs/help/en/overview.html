1.3 Overview of Functions

ORCS hat folgende Leistungsmerkmale:

  • Resource files of any size, not limited to 64K
     

  • Loading and saving of resource files with color icons
     

  • Loading and saving of resource file in intel format
     

  • Arbitrary number of files loadable (as long as memory suffices)
     

  • completely dnymically memory management
     

  • Trees and objects optionally in editor window or in own window
     

  • Names up to 32 characters
     

  • Configurable syntax of names (upper-/lowercase) for each resource
     

  • Possibility to test all tree types (alert, dialog, menu or image)
     

  • Arbitrary number of objects editable simultaneously (as long as windows are available)
     

  • Possibility to compare two resources
     

  • Search function for objects
     

  • Comments for objects and trees
     

  • Integrated icon editor (for color icons)
     

  • Possibility to load images in several formats
     

  • Standard GEM user interface
     

  • Works on all machines on which TOS or compatible systems run
     

  • Dialogs optionally in window or modal
     

  • Dialogs almost completely usable with keyboard
     

  • Loading and savging of von DEF, DFN, RSD, HRD and RSM definition files
     

  • Output files for C, Pascal, Modula, Basic, Fortran, Forth, Assembler etc.
     

  • Source output for C, Assembler and Pascal
     

  • Desktop optionally in background, in window or disabled
     

  • All flags & states of objects editable
     

  • WYSIWYG display of extended object types (checkboxes, radio buttons etc.)
     

  • Automatic alignment of menus items including shortcuts
     

  • Online-help for all dialogs
     

  • Interface optionally in english or german
     

  • Display of items optionally as icons or text
