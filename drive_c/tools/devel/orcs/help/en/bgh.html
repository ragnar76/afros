4.7 BubbleGEM Editor

The BubbleGEM editor is a special feature. Because here is something processed, which is not directly related to the resource. ORCS supports the export of BGH files. BGH files contain the texts for the bubble help "BubbleGEM". More information to BGH can be found in the BGH folder.

In the bubble editor the user-sections and the more-section can be edited. New entries are created by pressing the Insert-Key.
