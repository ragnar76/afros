3.1.1 Open

This item can trigger various actions.

  1. If there are one or more icons selected on the desktop they are opened as if you had made a double-click on it.
     

  2. If there is an item selected in the top window it will be opened as if you had made a double-click on it.
     

  3. If none of these conditions apply, the File Selector is called to load a resource file (Selectric, Boxkite, Freedom and all other file selectors creating a corresponding cookie are supported).
