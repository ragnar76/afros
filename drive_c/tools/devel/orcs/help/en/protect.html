3.4.1 Protection

Here you can set various levels of waht can be changed in a resource.

Protected Objects may be tested and viewed, but are protected against changes. Useful for example if you only want to view a file without to accidentally change something.
 
Locked Objects may be edited, sized or moved, but the object structure may not be changed. Usefull for example if you only want to make small corrections.
 
Beginner A warning is given before each update, that may affect an application.
 
Normal A warning is given befoe the workspace is cleared, or trees are rearranged. This is the default setting.
 
Expert No warnings are given when trees are altered.
