B File Types

The following file types are used in conjunction with ORCS:

ORCS.INF This file is created by ORCS when the current settings are stored. This file contains all settings, which can be made using the dialogues, as well as position and size of the icons on the desktop and all open windows. The settings are automatically reloaded when the program starts.
 
*.RSC The resource files themselves.
 
*.RSO The definition file with the names and comments of trees and objects for each resource file.
 
*.H Include-file for C.
 
*.RSH Source-Code for C.
 
*.S Include-file (and/or sourcecode) for Assembler.
 
*.I Include-file for Pascal.
 
*.RSI Source-Code for Pascal.
 
*.MOD Include-file for Modula (Implementation Module).
 
*.DEF Include-file for Modula (Definition Module).
 
*.LST Include-file for Basic.
 
*.INC Include-file for Fortran.
 
*.FOR Include-file for Forth.
 
*.BH Include-file for GFA-Basic 4.0.
 
*.RSD Definition-file for Kuma-Resource (NRSC).
 
*.DEF Definition-file for Ataris RCS (<= V1.4)
 
*.DFN Definition-file for Ataris RCS2 (>= V2.0)
 
*.HRD Definition-file for WERCS and Interface.
 
*.RSM Definition-file for ResourceMaster.
 
*.DUP Backup of resource files.
 
*.RSB Backup of definition files.
 

All names are only default values and may be changed at any time.

By use of modules other files may be generated.
