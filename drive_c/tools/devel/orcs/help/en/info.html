3.1.7 Show info

This item can (like "Open") trigger various actions.

  1. If there are one or more icons selected on the desktop. information to their corresponding object will be displayed.
     

  2. If there is an item selected in the top window, its information will be displayed.
     

  3. If there is an open window, information to its contents will be displayed.
     

  4. Otherwise the program information will be displayed.
