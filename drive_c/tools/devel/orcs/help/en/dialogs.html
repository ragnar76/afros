4.2.1 Object Dialogs

In all dialogs for editing objects, the object flags and states can changed and a name can be assigned. In the field 'Extended type' you can also specify a value between 0 and 255 which is written to the high byte of the ob_type field in the structure (see Extended Object Types.

Additionally, comments can be specified which are written to the corresponding output files.
